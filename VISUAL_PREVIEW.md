## ♿ 접근성 특징

### 스크린 리더 출력 예시

```
"2호선, 상행 열차, 3분 후, 상태: 정상, 강남역"

터치 가능한 경우:
"열차 상세 정보를 확인하려면 두 번 탭하세요"
```

### 색맹 대응

```
✅ 색상만으로 정보 전달 X
✅ 아이콘 + 텍스트 조합 사용
✅ 패턴 및 모양 차별화
✅ 충분한 대비 비율 (4.5:1 이상)
```

### 터치 영역

```
최소 크기: 44 x 44 points (iOS)
          48 x 48 dp (Android)

실제 카드 높이: ~100px
터치 가능 영역: 전체 카드
```

## 📊 성능 메트릭

```
렌더링 시간:     < 16ms (60fps)
```
